SecuritiesLook is your guide to security tokens. We aim to provide you a complete information about at all Security Tokens and Security Token Offerings (STOs) Worldwide at one place.


Security Tokens are the digital representation of a physical value that are stored in a Blockchain. Security Tokens can represent any investment types such as Company, Bond, Equity, etc..,

In other words, Security tokens are digital representations of traditional securities such as equities, bonds and options.


Security Token Offerings is a regulated fund raising methodology were the investor can purchase Security Token Offered by issuer or owner.


We provide you the informations about Security Token Offerings (STOs) and Security Tokens.

The informations about Security Token Offerings (STOs) includes Name of the project and their Sale status, Investment Type, Project Industry, Location, Company Size, Age.

The informations about Security Tokens includes Name of the Security Tokens and their Price, Market capitalisaton, Price change (24hrs), Volume (24hrs) , Graph (7day).
